Quickstart

Index with POST /reindex, then use POST /search and POST /open.

curl -X POST http://127.0.0.1:8002/reindex
curl -X POST http://127.0.0.1:8002/search -H "Content-Type: application/json" -d '{"query":"reindex"}'
